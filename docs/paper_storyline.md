# Paper Storyline

## Main Argument

Solid-state electrolyte dopant research should not optimize a single conductivity number in isolation. A performance-aware data chain can reveal dopant rules that better connect materials chemistry to practical battery behavior.

## Storyline

1. Solid-state electrolyte doping studies often overemphasize ionic conductivity.
2. A literature database is built around the composition-processing-transport-interface-battery chain.
3. PERD/BPRS discovers performance-relevant dopant rules.
4. Temporal extrapolation tests whether early literature predicts later literature.
5. A small number of counterfactual experiments validate whether algorithmic ranking matches cell-level performance.

## Possible Journals

Potential journals include `npj Computational Materials`, `Digital Discovery`, `Energy Storage Materials`, `Advanced Functional Materials`, and `Advanced Energy Materials`. Fit depends on database scale, algorithm performance, and experimental validation strength.
