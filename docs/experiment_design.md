# Experiment Design

This repository does not execute experiments. It prepares candidate rankings for later counterfactual validation.

## Counterfactual Set

- `S0`: baseline composition or known reference.
- `S1`: high BPRS candidate 1.
- `S2`: high BPRS candidate 2.
- `S3`: medium BPRS candidate.
- `S4`: low BPRS counterexample.

## Required Characterization

- XRD.
- SEM.
- Relative density.
- EIS.
- Temperature-dependent EIS.
- Li symmetric cell.
- Critical current density.

## Core Validation Criterion

The key test is whether PERD/BPRS ranking is closer to real battery-performance ranking than conductivity-only ranking.
